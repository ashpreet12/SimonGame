Rules...

  • Press the START Button to start the game. SIMON will give only the first signal.
  • Repeat Simon's signal, and immediately SIMON gives another signal.You will have to remember the Sequence!
  • Try to reach as far as you can!
  • If you make an error to play any signal, you'll hear the "RAZZ" sound. This means the game is over.
  • To play again after the game ends simply press the RESTART Button to restart the game.
